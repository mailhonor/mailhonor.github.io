honor-grid 在linux64位环境编译运行, C开发.

编译的目标是得到安装包honor_grid.tgz,也可以直接下载安装包:稍等...

依赖的库

mysqlclient, ssl, pam, 等

下载源码

git clone https://github.com/mailhonor/honor-grid.git
cd honor-grid

编译过程

make

打包 

make tgz

得到 honor_grid.tgz
